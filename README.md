# cnlp

## Requirements

* Python 3

## Commands

```bash
$ python -m venv .py3env # 创建venv环境
$ source .py3env/bin/activate # 进入venv环境
(venv) $ pip install -r requirements.txt # 安装依赖
(venv) $ pip install scipy # 安装一个包
(venv) $ pip freeze > requirements.txt # 保存安装包清单
(venv) $ deactivate # 退出venv环境
```
